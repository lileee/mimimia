# 素材来源

## 结论

网页可见的角色、灵猫、音乐、音效和程序画面均为本项目原创或由原创源文件导出。没有加入来源不明的影视、游戏或网络采样，也没有分发第三方字体、纹理或图标文件。

唯一完整、可机器核对的记录是 [`docs/assets/asset-ledger.csv`](assets/asset-ledger.csv)。每条记录包含素材编号、路径、类别、创作者或来源、制作方式、修改、权利依据、公开状态和 SHA-256 校验值。

## 图像

| 组别 | 原始内容 | 处理与公开形式 |

原创图片提示保存在 `art/prompts/`，母图和图层保存在 `art/source/`，人工与自动审核图保存在 `art/reviews/`。

## 声音

全部声音由项目脚本以确定性加法合成生成：

- 月光虚境环境音乐；
- 三层蓄力循环；
- 蓄满提示、失败消散、释放铃声和灵猫成形音效。

无损 WAV 保存在 `art/audio/source/`，网页使用的 MP3 保存在 `public/assets/audio/`。波形、时长、峰值、循环点和触发时机见 `art/reviews/audio-cue-sheet.md`。

## 字体与依赖

界面只声明系统字体回退列表，不随仓库分发字体文件。Three.js 等程序依赖通过 `package-lock.json` 固定版本并保留各自许可证；它们不作为原创素材登记。

## 私有边界

本地灵感参考只用于早期方向理解，未进入公开素材、源代码、截图、台账或交付包。`npm run assets:private-check` 与 `npm run release:verify` 会同时检查路径和内容副本。
