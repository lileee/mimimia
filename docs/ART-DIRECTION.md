# 美术方向

## 核心气质

画面是安静、清澈的月光仪式，而不是高饱和战斗特效。主体使用银白、深靛、淡紫和少量冷金；最亮区域只落在法阵中心、法杖月轮和灵猫显形边缘。

## 魔法少女

- 银白长发、宽檐星月帽、深靛披肩与白金法衣构成远距离识别轮廓。
- 人物保持轻微俯视下的完整全身构图，帽尖和鞋底必须留在安全画面内。
- 等待状态只做呼吸、发丝、披风、裙摆和法杖的轻微错相运动，避免像纸片整体漂浮。
- 脸部和胸口是保护区，粒子不得形成持续遮挡或过曝。

## 月辉灵猫

- 体型小于人物头肩范围，以银白软毛、紫色耳尖和月牙尾巴识别。
- 从人物脚边的深紫轮廓开始，由下向上填充银蓝星光，完成后移动到右肩上方。
- 灵猫不能挡住人物脸部和法杖月轮；完成后保留呼吸、漂浮、眨眼、尾巴摆动和鼠标视线响应。

## 母图与拆层

少女和灵猫各自先确定一张完整原创母图，再从同一母图拆层。拆层不重新生成不一致的部件。缺失的接缝、颈部、发丝、裙摆与尾根遮挡由同一母图补画，并保留透明边缘。

少女源文件使用统一 3150×6144 画布，运行时缩放为 1575×3072；灵猫源文件使用统一 3072×3072 画布，运行时缩放为 1536×1536。枢轴和前后顺序写在各自 `rig.json` 中。

## 防穿帮检查

- 每次修改图层后运行 `npm run art:validate-layers`。
- 使用 `npm run art:render-extremes` 生成 1× 与 2× 极限位移板。
- 在黑、白、深紫三种背景检查白边、空洞、错层和接缝。
- 浏览器中再检查 1440×900 与 1920×1080 两种视口，不能只看源图。

原创提示、母图、图层、审核记录和运行时导出之间的对应关系见 [素材来源](ASSET-SOURCES.md) 与 `docs/assets/asset-ledger.csv`。
