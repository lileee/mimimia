# 性能与稳定性实测报告

记录时间：2026-07-14（中国标准时间）

## 结论

- 本机正式版 Chrome 的 WebGPU 高画质与强制 WebGL 2 兼容画质都完成了 1920×1080、稳定 10 秒、连续 3 次召唤；平均帧率均超过 118，最长停顿低于 85 毫秒。
- 连续 20 次召唤与重置后，对象数量没有增长；再静置 30 秒，内存回落到 23.67 MiB。
- 画质选择、连续低帧降级、演出期间延后、30 秒冷却和窗口隐藏清空均通过自动检查。
- 当前实测机器是 Apple M5 Pro、48 GB 内存，性能高于计划指定的 Apple M1 / Intel Iris Xe 同级普通设备。因此本报告不把“普通设备基准”标为最终通过；需在公开页面完成后补一台目标档设备实测。

## 画质规则重放

2026-07-14 执行 15 项画质检查，全部通过：

| 输入或场景 | 预期与实测结果 |
| --- | --- |
| 45 帧 | 高画质 |
| 30 帧 | 均衡画质 |
| 29.9 帧 | 兼容画质 |
| 单个低于 27 帧的 5 秒窗口 | 不降级 |
| 连续两个低于 27 帧的窗口 | 请求降低一档 |
| 蓄力、蓄满、消散、召唤中 | 暂缓切换，回到安全状态后执行 |
| 冷却 | 两次自动切换至少相隔 30 秒 |
| 标签页隐藏、失焦或图形异常 | 清空不完整样本 |
| 手动固定画质 | 不参加自动降级 |
